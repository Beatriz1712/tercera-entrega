### 💻 Instalación:


1. Se inicia el proyecto.
```shell
src/index.js
```
2. Instalar las dependencias para el servidor.
```shell
npm i express 
npm i //dependencias
```


### 🌐 Servidor:

Se levanta el servidor con *Express js* en el puerto:

#### `http://localhost:8080`

Para iniciar la aplicación, ejecuta el siguiente comando:
```shell
npm start
```
